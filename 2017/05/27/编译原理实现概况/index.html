// 三种形式：
// 1.无参数(形参声明为void。如getc等)
// 2.定长参数(参数的个数是相同的。如puts或fgets等)
// 3.可变长参数(参数的个数不确定。如printf等)
注意：
//定长参数或可变长参数
//C语言必须有1个或1个以上的形参后才支持可变长参数
//f(…)是不允许的，至少要f(int x, …)
//原因在于C语言是通过可变长参数的前一个参数的地址来取得剩余参数的
